# CryptoWebsite

In order to run this you will need to add the appsettings.json file in /CryptoWebsite

{
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft.AspNetCore": "Warning"
    }
  },
  "AllowedHosts": "*",
  "ConnectionStrings": {
    "CryptoWebsiteContext": "{yourconnectionString}"
  }

}
